# HabitTracker – Seguimiento de Hábitos


## Opción elegida
🏋 Seguimiento de Hábitos


## Cómo ejecutar la app
1. Clonar el repositorio
2. Ejecutar: npm install
3. Ejecutar: npx expo start
4. Escanear el QR con Expo Go o presionar 'a' para Android


## Funcionalidades implementadas
- Registro e inicio de sesión con AsyncStorage
- Agregar hábitos con nombre y frecuencia (Diario/Semanal)
- Listar hábitos y marcarlos como cumplidos
- Eliminar hábitos
- Notificación local a los 10 segundos
